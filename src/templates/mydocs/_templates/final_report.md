## PR 검토 요청 (인간 게이트 2)

- charter 전 수용 기준이 OK이므로 자율 LOOP를 종료하고 `{BASE_BRANCH}` 대상 Open PR을 게시한다.
- loop-state는 `awaiting_merge`로 남겨 CI가 이 PR의 charter를 계속 권위 대상으로 해소한다. `done`은 merge 후 GitHub `MERGED` 사실에서만 도출한다.
- PR 검토와 merge는 작업지시자가 결정한다(ultra-waterfall의 마지막 인간 개입).
